# GRG⚙️
GRG is a instant github readme generator.
> [!IMPORTANT]
>**Don't forget to replace `username` with your github username.**

## Features✨
- Instant generation
- User friendly design

## Technologies used💻
- HTML
- CSS
- JS

## Goal🎯
GRG is created so that user can create reamde in no time.
